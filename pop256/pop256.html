- INSTRUCTIONS MANUAL -

A program line looks like :

COMMAND + SPACE + VALUE
ex: PUT 10
(value is not needed in some cases)

Press /ENTER/ to validate the line.
Press /LCTRL/ to execute.
Press /LCTRL/ again to go back to text mode.

Instruction set :

PUT value (push value to stack)
POP (pop last value and push it on return stack)
PUL (pop last return value and push it on stack)
ADD (replace first two values with sum)
SUB (replace first two values with difference)
MUL (replace first two values with product)
DIV (replace first two values with quotient)
JNR value (jump to line if not = to return)
DUP (duplicate top stack)
DIS (draw to screen, top = x, second = y)
SND (play sound, top = note, max 83)
CLR (clear screen)
